# geo
#v掌柜地理位置与网格经理绑定功能实现
##实现原理：
1. 获取关注粉丝的地理位置
2. 通过百度地图API获取实际地址
3. 判断该粉丝是否已经绑定客户经理
4. 未绑定则利用实际地址与明星掌柜的地址信息匹配，找出最近的一个。
5. 推送该明星掌柜的信息给当前粉丝，包括但不限二维码，V店信息，个人介绍等。

##其他事项
1. 需要百度地图API做经纬度与地址的转换
2. 此功能会通过后台绑定明星掌柜,因此需要提供明星掌柜的具体信息,包括但不限客户经理ID,姓名,V店链接,详细地址,所属区域等.
